# Superstore Business Intelligence Dashboard  
*A Streamlit-based Interactive BI System for Sales, Profit & Performance Analysis*

---

##  Overview
This project presents an interactive **Business Intelligence Dashboard** developed using **Python, Streamlit, Plotly**, and **Pandas**.  
The dashboard provides meaningful insights into the **Superstore dataset**, enabling users to explore sales patterns, profit distribution, regional performance, and monthly trends.

It is designed to support **data-driven decision-making** through clean visualisations and dynamic filtering.

---

##  Key Features
1 Interactive KPI Summary (Sales, Profit, Total Orders)  
2 Sales by Region (Bar Chart)  
3 Profit Distribution by Category (Donut Chart)  
4 Monthly Sales Trend (Line Chart)  
5 Sidebar Filtering (Region, Category, Date Range)  
6 Clean and Responsive Streamlit UI  
7 Ready for future extensions (customer segmentation, predictive analysis, etc.)


---

##  Technologies Used
- **Python 3**
- **Streamlit** for UI & dashboard
- **Pandas** for data handling
- **Plotly** for interactive charts
- **Altair** (optional) for advanced visuals
